========

An Ultimate BMS Emulator Project (Rhythm Game)

작성일 2/2 기준 현재도 작업중입니다.

개발 언어는 C++/C 입니다.
주력 기능은 BMS Emulator이고, Lunatic Rave 2와의 일부 호환(스킨)을 특징으로 삼고 있습니다.
기존 에뮬레이터들과 차별화된 이 에뮬레이터의 특성은 다음과 같습니다. (예정)
(중요도 순서대로 배열)

Archive File(zip, rar) 지원을 통한 게임 파일 분산화 방지
모바일 기기로도 포팅, 다양한 플랫폼에서도 즐기는 리듬게임
고수도 입문자도, 컴퓨터에서도 모바일에서도 플랫폼과 실력을 넘어 모두가 재미있게 즐길 수 있는 멀티플레이 시스템 구현 (상황에 따라 이 부분은 일부 상용화가 이루어 질 수 있습니다)
(단순한 편의성 용도) BMS 간이 서버/클라이언트 기능 지원
프로젝트의 작업기간은 매우 길 예정입니다. 기본적인 플레이 기능 구현만 해도 당장 한두달의 개발기간을 염두에 두고 있으며, 기능 전체의 구현은 적어도 6개월, 길면 몇년의 시간이 걸릴지 알 수 없습니다.

진척이 굉장히 느린 관계로 (^^;) 별도로 백업 브랜치를 같이 구동할 생각입니다.
뛰어난 개발자분들의 적극적인 피드백 및 커밋을 기다립니다.

Development Language: C/C++
Rhythmus is a BMS Emulator, and it has a little compatibility with Lunatic Rave 2 (Skin)
It has some specialized features like these -

Supports Archive FIle (zip, rar)
Mobile device support, Groove at anywhere you want!
Multiplay Network System which many people can enjoy - From Beginner to Expert, From Computer to Mobile. (Some features may be commercial)
(For Convinence) BMS mini Server/Client feature
Capacity for this project will be very long - It may be need one or two months for Module for play, and total may need a few years.
